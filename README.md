This is based on a basic Newton mechanics, It evauated of velocity using this formulate:
V=Δx/Δt
So ---> x=distance; t=time evaluate in 2 points as this table:
----------------------
time      |  distance
----------------------
    0     |    2
    1     |    5
    2     |    8
    3     |    11
    4     |    14
    5     |    17

if we applied this, so the final velocity would be:
V=17-2/5-0=15/3=3m/s
this is accurate to the graphics.
